# Tourism Package Prediction

This project predicts whether a customer will purchase a tourism package.

## Features
- Machine Learning model using XGBoost
- SMOTE for handling imbalance
- Streamlit app deployment
- Hugging Face model hosting
- CI/CD using GitHub Actions

## Inputs
- Age
- Monthly Income
- Number of Trips
- Passport
- City Tier

## Output
- Purchase Probability
- Final Prediction
